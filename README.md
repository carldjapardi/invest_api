# Invest API

Micro transactions with RAG and web search for agentic automation connected to broker, bank, and host systems.

## Features

- Paper trading and real-money modes for testing strategies before enabling live trades.
- Configurable risk profiles that define trade size, max loss, diversification, and approval requirements.
- Human-in-the-loop controls for reviewing, approving, or blocking automated trading decisions.
- Brokerage and bank connections for account access, funding context, and trade execution.
- RAG-powered market context that can retrieve relevant portfolio, strategy, and research information.
- Web search support for gathering current market, company, and macroeconomic signals.
- Trade nuance settings for daily trade limits, model selection, asset preferences, and strategy constraints.

## User Process

1. Choose paper trading or real-money mode.
2. Define risk profile and amount of human intervention.
3. Set up connections to brokerage and bank accounts.
4. Configure trade nuances, including amount of trades per day, max loss, diversification, and model.
